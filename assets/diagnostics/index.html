Run MTR from Windows (WinMTR)

    

      Test the network path from your PC to this server at
      __SERVER_DOMAIN__
      (IP: __SERVER_IP__).
    

    
    
      
  1. 
        Download WinMTR from the official site:
        
          github.com/White-Tiger/WinMTR
        — extract the ZIP and run WinMTR.exe.
      
    2. 
      
  2. 
        In the Host field, enter either the server IP
        (__SERVER_IP__)
        or the domain
        (__SERVER_DOMAIN__).
      
    3. 
      
  3. 
        Set Max hosts to 30 and
        Interval to 1000 ms.
      
    4. 
      
  4. 
        Click Start and let it run for at least 300 packets (about 5 minutes).
      
    5. 
      
  5. 
        Click Export TEXT to save the results, then share them with your support team.
